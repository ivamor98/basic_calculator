# Калькулятор
## 1 Описание проекта
Калькулятор осуществляет операции сложения, вычитания, умножения и деления двух чисел. Поддерживает отрицательные и дробные числа.
## 2 Описание стека используемых технологий.
- Qt6 - версия кроссплатформенного фреймворка Qt для создания графических приложений для десктопа, мобильных и встроенных устройств.
